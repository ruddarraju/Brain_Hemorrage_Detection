# 🧠 AI-Driven Brain Hemorrhage Detection using Quantum-Optimized BiLSTM

Early detection of **brain hemorrhage** is crucial for saving lives. Delays in diagnosis can lead to severe neurological damage. This project explores the use of **Artificial Intelligence and Quantum-inspired optimization** to assist in faster and more accurate hemorrhage detection from **CT scan images**.

Our system combines **deep learning (BiLSTM)** with **quantum-inspired optimization techniques** to improve feature learning and model efficiency. The proposed model analyzes patterns from medical imaging data and predicts the presence of hemorrhage with high reliability.

The model achieved an **accuracy of 98.9%**, demonstrating the potential of combining advanced AI architectures with optimization techniques for medical diagnostic support.

---


## 🔬 Project Motivation

Brain hemorrhage is one of the most dangerous neurological emergencies. Diagnosis typically relies on expert radiologists analyzing CT scans, which can be time-sensitive and resource-intensive.

This project aims to build an **AI-assisted diagnostic tool** that can:

* Support early detection of hemorrhage
* Reduce diagnostic workload for medical professionals
* Improve reliability of automated screening
* Explore the potential of **quantum-inspired AI techniques in healthcare**

## ⚙️ How the System Works

1. **CT Scan Image Input**
   Medical CT images are used as the primary data source.

2. **Image Preprocessing**
   Noise reduction, normalization, and feature extraction are applied to prepare the data.

3. **BiLSTM Model Processing**
   A **Bidirectional Long Short-Term Memory network** learns complex spatial patterns and dependencies within extracted features.

4. **Quantum-Inspired Optimization**
   Optimization techniques inspired by quantum computing help improve training efficiency and model performance.

5. **Prediction Output**
   The system classifies whether the CT scan shows signs of **brain hemorrhage or normal condition**.

## 🚀 Running the Project

1. Clone the repository

```
git clone https://github.com/ruddarraju/brain-hemorrhage-detection.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python main.py
```

4. Upload a CT scan image and the system will generate a prediction.
